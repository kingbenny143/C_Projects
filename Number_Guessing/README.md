# Number Guessing Game

A console-based number guessing game built in C.

## Features

- Random number generation
- User input handling
- Attempt counter
- High/Low hints
- Win condition

## Concepts Used

- Random Numbers
- Loops
- Conditional Statements
- Functions
- Problem Solving

## How to Run

```bash
gcc main.c -o guessing_game
./guessing_game
```
