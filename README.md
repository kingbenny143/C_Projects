## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/C-Programming-Projects.git
```

Navigate to a project folder:

```bash
cd project-folder
```

Compile the program:

```bash
gcc main.c -o program
```

Run the program:

```bash
./program
```
